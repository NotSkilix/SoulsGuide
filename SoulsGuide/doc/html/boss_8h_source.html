Soul’s Guide is a retro arcade side-scroller game inspired by Super Mario Bros. (1985), integrating modern rogue-like elements (death returns the player to the beginning of the game, with progressive difficulty) and speedrun features (the goal is to complete the game as quickly as possible). It combines classic platformer mechanics with modern functionalities, providing a gaming experience that is both stylized, nostalgic, and innovative.
